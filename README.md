# Cloud Resource Allocation Using Greedy Algorithms

## Overview

This project evaluates custom greedy load-balancing algorithms for cloud resource allocation using Java and CloudAnalyst.

The objective is to improve virtual machine (VM) utilization and reduce response times by intelligently assigning incoming workloads across cloud resources.

## Algorithms Implemented

* MinLoadGreedy
* ValueRatioGreedy
* EarliestAvailableGreedy
* CapacityGreedy

## Benchmark Algorithms

* Round Robin
* Particle Swarm Optimization (PSO)
* Honey Bee Optimization
* Throttled Load Balancing

## Evaluation Metrics

* Response Time
* Processing Time
* Resource Utilization
* Load Distribution Efficiency

## Key Results

* Improved resource utilization by approximately 30%.
* Reduced workload imbalance across virtual machines.
* Demonstrated performance trade-offs between greedy and swarm-based optimization methods.
